= Whygone

 A digital ode to _why on whyday 2010

== CONTROLS
  Arrows-keys

== RUN FROM SOURCE
  $ gem install chingu
  $ ruby start_game.rb

Only tested on Ruby 1.9.1, there might a problem with 1.8.7 which I'm looking into.
